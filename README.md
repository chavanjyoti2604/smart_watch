# Smart-Digital-Watch
Smart Digital Clock is a web-based application developed using HTML, CSS, and JavaScript. It displays the current time in a live 12-hour format (HH:MM:SS AM/PM) and updates every second using JavaScript's Date() object and setInterval() function.

The application includes several interactive features such as a Dark/Light Mode Toggle, Dynamic Background Changes based on the time of day (Morning, Afternoon, and Night), a Countdown Timer to New Year, and an Alarm System that allows users to set a specific time and receive an alert when that time is reached.

This project demonstrates key front-end development concepts including DOM Manipulation, Event Handling, Real-Time Updates, and Responsive UI Design, making it an excellent beginner-level JavaScript project.
